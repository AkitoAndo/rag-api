🖼️ RAG API - 画像処理機能拡張

既存のRAGシステムに画像保存・OCR・ナレッジベース統合機能を追加

🎯 核心コンセプト: 画像 → OCR → ナレッジベース統合

1. 画像保存 → 2. 文字化（OCR） → 3. 既存ナレッジに投稿

ユーザーがアップロードした画像から自動的にテキストを抽出し、既存のテキストベースRAGシステムと統合してシームレスな検索・回答機能を提供します。

🖼️ 画像管理機能

  • JPEG/PNG/GIF/WebP対応アップロード
  • S3での安全な画像保存
  • タイトル・説明・タグ管理
  • マルチテナント分離

🔤 OCR・文字化機能

  • Amazon Textractによる高精度OCR
  • 日本語・英語対応
  • 表・図表からのテキスト抽出
  • 信頼度スコア付き結果

📚 ナレッジベース統合

  • OCRテキストの自動ベクトル化
  • 既存文書との統合検索
  • ハイブリッド（テキスト+画像）クエリ
  • コンテキスト保持回答生成

🔐 セキュリティ・クォータ

  • Cognito JWT認証継承
  • ユーザー別データ完全分離
  • 画像数・ストレージ制限
  • プラン別機能制御

🚀 開発者向け情報

Cognitoトークンの設定: ブラウザの開発者コンソールで以下を実行してテスト用トークンを設定できます：

setCognitoToken('your-jwt-token-here')

API基本URL: https://your-api-gateway-url.amazonaws.com/prod

認証方式: Bearer JWT (Amazon Cognito)
